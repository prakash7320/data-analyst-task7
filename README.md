# Task 7 - Basic Sales Summary using SQLite + Python

This project demonstrates how to connect Python with a SQLite database,
run SQL queries, load the results into pandas, and visualize revenue using a bar chart.

## Steps Performed:
1. Created a SQLite database (sales_data.db)
2. Added a "sales" table with sample product data
3. Wrote SQL query to calculate:
   - Total quantity
   - Total revenue (quantity * price)
4. Displayed summary using print
5. Created a bar chart using matplotlib
6. Saved output chart as sales_chart.png
